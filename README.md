# StutteredBars
Easy C# BARS parser targeting version 5 (Splatoon 3, Tears of the Kingdom, Mario Wonder)
# Usage
```cs
BARSFile bars = new BARSFile(File.ReadAllBytes("BgmVersusFest_SAND.bars"));
```
